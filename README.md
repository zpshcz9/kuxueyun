低价LXC容器不提供技术支持，如果你是纯新手，建议从KVM系列入手，带管理面板操作更方便。

如果你用信用卡或支付宝付款，记得会有3%+$0.3的Stripe通道费。小额订单的话这个比例会显得高一些，年付大套餐的话就基本可以忽略了。
